Helm Repository

MCP Orchestrator

Kubernetes-native control plane for Model Context Protocol servers.

Install with Helm

# 1. Add this repository
helm repo add magertron https://magertron.com/charts
helm repo update

# 2. Install into a new namespace
helm install mcp magertron/mcp-orchestrator \
  --namespace mcp-system --create-namespace
